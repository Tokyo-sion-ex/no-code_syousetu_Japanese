# 【挨拶】
ようこそ、ノーコード小説へ。
ここでは、Github＝コード投稿 という概念を突破し(?)、小説を読んでもらえるようにしたい！という目論見で誕生しました。
誰でも小説を投稿できるようにしています。
また、ファイル作成が可能であればユーザーごとにファイルを作成すると尚便利です。

是非皆さんも使ってみて下さい〜

# 【基本的ルール】
1. 許可なく他人の作品を加工・削除しない
2. 作成は自由
3. 問い合わせフォームを利用する際は必ず1トピックごとにタイトル(#)で区切ること
4. 小説のテーマは不問。
5. 性的表現、暴力表現、残酷表現が含まれる場合は必ず明記すること。
  5-1. 詳細は「特筆表現について」をご閲覧下さい。
# 【特筆表現について】
《性的表現》
BLやセンシティブ、その他前述に属する内容を指します。
R16やR18で性的表現が含まれる際は**必ず**明記して下さい。
《暴力表現》
人間同士の戦闘シーンが含まれているシーンを指します。
R16(場合によってはR18)で暴力表現が含まれる際は**必ず**明記して下さい。
《残酷表現》
流血、死亡シーンが含まれる内容を指します。
R16(場合によってはR18)で残酷表現が含まれる際は**必ず**明記して下さい。

# 【削除に関するルール】
以下の状態が確認された際には警告をした後、確認されなかった際は削除を行いますのでご了承下さい。
・特筆表現に該当するにも関わらず明記をしていない。
・特定の人物に対する名誉毀損行為
・事実無根の内容
・宣伝行為
・デモの呼びかけ

# 【許可されている行為】
・時事問題に対する一線を越さない批評
・日記(ブログ含む)
・技術の解説

# 【その他】
その他でご不明な点がございましたら「問い合わせフォーム」までご相談下さい。
